Почему Dart?

Разработчики в Google и все остальные используют Dart для создания высоко-качественных приложений и решения высоко-критичных задач для iOS, Android и web. С возможностями нацеленными на клиентскую разработку, Dart отлично подходит как для мобильных приложений, так и для web.

Dart свободный и открытый, разрабатывамый на GitHub.

Продуктивный

Синтаксис языка Dart чист и краток, его инструментарий простой и мощный. Звучная типизация помогает вам обнаружить тонкие ошибки заблаговременно. Dart имеет закалённую в бою стандартную библиотеку и экосистему из тысяч пакетов.

Быстрый

Dart предоставляет оптимизированную AOT компиляцию, чтобы получить предсказуемо высокую производительность и быстрый запуск на различных устройствах и на web.

Портативный

Dart компилируется в архитектуры ARM и x86, чтобы мобильные приложения на Dart могли запускаться нативно на iOS, Android и за их пределами. Для web приложений Dart транспилируется в JavaScript.

Доступный

Dart покажется знакомым большинству существующих разработчиков, благодаря объектной ориентации без сюрпризов и синтаксису. Если вы уже знаете C++, C# или Java, вы можете быть продуктивным с Dart уже через несколько дней.

Реактивный

Dart полностью подходит для реактивного программирования с поддержкой управления коротко-живующими объектами, такими как UI виджеты, с помощью быстрого выделения объектов и сборщика мусора. Dart поддерживает асинхрнонное программирование с помощью возможностей языка и API, которые используют Future и Stream объекты.
