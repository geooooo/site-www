Dart - это клиент-оптимизированный язык для быстрых приложений на любой платформе
Оптимизирован
для UI
Разработка на языке программирования, заточенном под нужнды создания пользовательских интерфейсов
Продуктивная
разработка
Вносите изменения итеративно: используйте горячую перезагрузку (hot reload), чтобы мгновенно видеть изменения в вашем запущеном приложении
Быстрый на всех
платформах
Компилируется в машинный код ARM & x64 для мобильных платформ, desktop и backend. Или компилируется в JavaScript для web
Оптимизирован для UI
  • Зрелый и полноценный async-await для пользовательских интерфейсов, содержащих событийно-ориентированный код, в паре с базирующейся на изолятах конкурентностью
  • Язык программирования оптимизирован для создания пользовательских интерфейсов с такими возможностями как spread оператор для распаковки коллекций, и if для коллекций, чтобы адаптировать UI для каждой платформы
  • Язык программирования, который прост в изучении, c знакомым синтаксисом
Продуктивная разработка
  • Вносите изменения в ваш исходный код итеративно, используя горячую перезагрузку (hot reload) чтобы мгновенно увидеть изменения в запущенном приложении
  • Пишите код, используя гибкую систему типов с богатым статическим анализом и мощным, настраиваемым инструментарием
  • Профилируйте, логируйте, и отлаживайте с выбраным вами редактором кода
Быстрый на всех платформах
  • AOT-компиляция приложений в нативный машинный код для мгновенного запуска
  • Используйте для web с полноценными, зрелыми, быстрыми компиляторами для JavaScript
  • Запускайте backend код необходимый вашему приложению, путем написания на одном языке программирования
